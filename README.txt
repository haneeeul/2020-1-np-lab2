How to use:
	if you use 'gcc', then you have to add this option '-lpthread'.
	
	gcc -o [name] -Wall -lpthread webserver_linux.c

	and execute the .exe file with port number.

	Go to web browser, enter below line in address field.

	http://localhost:[PORTNUMBER]/[NAME].html 

	Then you can see the result on both browser and command line.
